# MirrorFit Showcase

MirrorFit is an AI personal stylist product prototype.

The project explores how AI image generation can help users transform their personal photos into styled fashion visuals based on selected aesthetics.

## Product Concept

Upload photo → Choose vibe → Generate styled look

MirrorFit focuses on:

- AI personal styling
- Vibe-based image generation
- Identity preservation
- Fashion-oriented UX
- Lightweight product flow

## Current Vibes

- Clean Girl
- Soft Feminine
- Streetwear
- Night Luxe
- Y2K Pop

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Python
- AI image generation APIs
